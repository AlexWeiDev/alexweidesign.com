Description

Chron is an ecommerce website template designed for watch shop, it provides a series of functions, paring with elegant layout and style, to fulfill various needs from customer. It is highly responsive to most of devices such as desktop, tablet and mobile phone so that customer can do shopping anytime and anywhere.

Tools

HTML / CSS / JavaScript / Axure / Photoshop / Illustrator / Git

Skills

Axure / illustrator / UI & UX design / Front end Programming

Design Process

Stage 1: Ideation

We started this project with a lot of research and discussion toward its design context, scrope and primary user because what’s we decided and mapped here sets the outline for the entire project. According to the design philosophy behind product of Chron, we plan to build this template by comprising elegant visuals and layout that can brings easy understanding user flow and the value of brand to customer. Check the style guide in website for more information.In order to guide the design flow and reduce designer's confusion, we also worked out a sitemap to show the possible user flow of website.

Stage 2: Mockup

Before going into more detailed prototype, we designed the layout and build some static mockups by using information gathered from the planning phase so that our designer can quickly test some designs and replace it if it will be too challenge to us in coding phase. These mockup keeps the design elements organized in layers and it illustrates different structures of website when it responses to different devices. It helps to prime our programmer for developing when the time later on.

Stage 3: Development

When all design and contents are settled, we began to code each designed mockup page into HTML web page. Since the website should be perfectly responsive to different device without losing any feature When user resize, hide and shrink the screen, we put effort into choosing suitable coding method and finally applied the responsive grid (fluid grid structure), which can automatically resize contents and screen according to specific ratio, to build our web pages.
